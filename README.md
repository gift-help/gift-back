# Gift Help Backend

## Быстрый старт

1. Установить зависимости:
   ```bash
   npm install
   ```
2. Запустить проект в режиме разработки:
   ```bash
   npm run start:dev
   ```

## Swagger документация

После запуска проект будет доступен по адресу:
- Swagger: [http://localhost:5000/api/docs](http://localhost:5000/api/docs)

## Тестовый запрос

Для имитации работы запроса с задержкой 5 секунд используйте эндпоинт:
- `GET /temporary/generate-gifts`

Этот запрос возвращает список подарков через 5 секунд.

---

**Требования:**
- Node.js 22.18.0
